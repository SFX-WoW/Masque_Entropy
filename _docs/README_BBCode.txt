[COLOR=Orange][SIZE="4"]About[/SIZE][/COLOR]

[I]Entropy[/I] is an add-on for for [URL="https://worldofwarcraft.com"]World of Warcraft[/URL] that adds a set of skins for [URL="https://www.wowace.com/projects/masque"]Masque[/URL]. This set is based on [URL="https://www.wowace.com/projects/masque-apathy"]Apathy[/URL] and includes a number of metallic skins.

[I][B]Note:[/B] You must have [URL="http://www.wowinterface.com/downloads/info12097"]Masque[/URL] installed to use this add-on.[/I]

[COLOR=Orange][SIZE="4"]Skins[/SIZE][/COLOR]

[COLOR=Orange][SIZE="4"]Installation[/SIZE][/COLOR]

1. Download the package from one of the following sites:
   [LIST]
   [*] [URL="https://mods.curse.com/addons/wow/masque-entropy"]Curse[/URL]
   [*] [URL="https://www.wowace.com/projects/masque-entropy"]WoW Ace[/URL]
   [*] [URL="http://www.wowinterface.com/downloads/info8873"]WoW Interface[/URL]
   [/LIST]
2. Extract the package to your "World of Warcraft\Interface\AddOns" directory.
